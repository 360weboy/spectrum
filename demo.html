Spectrum - Drop in colorpicker

No images

Spectrum requires one JavaScript file and one CSS file. No images required.

Browser Support

Browser support depends upon CSS gradient support. IE is handled using proprietary filters.
  • IE 6+
  • Chrome 4+
  • Firefox 3.6+
  • Safari 4+
  • Opera 11.1+

Progressive Enhancement

You can call spectrum() on any element, but it is recommended to call it on an input element. Spectrum will hide the input, then automatically copy the selected value back into the input as it changes, so you don't need to worry about that. If you are using the results from a form, the server will see the selected color as if the user typed in the hex code manually.

Usage modes

Flat versus Collapsed

  • Flat: You can use it flat (see the picker to the left). This means that it will always show up at full size, and be positioned as an inline-block element.

Skinning

Since it is all built with HTML/CSS, you can skin it easily. There are two parts to the spectrum.css file, the core rules (at the top of the file), and the themable rules (at the bottom). Feel free to tweak these rules to make it look how you want.
