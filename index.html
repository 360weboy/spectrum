Spectrum - The No Hassle Colorpicker

Source code and downloads | Me

Goals

I wanted a JavaScript colorpicker that didn't require images, and that had an API that made sense to me as a developer who has worked with color in a number of applications. I had used existing plugins (which I was quite grateful for), but decided that I would make a smaller, simpler one. Here are the goals I had:

Drop In

Spectrum requires one JavaScript file and one CSS file. No images or other resources required. This makes it easy to add into projects.

Progressive Enhancement

You can call spectrum() on any element, but it is recommended to call it on an input element. Spectrum will hide the input, then automatically copy the selected value back into the input as it changes, so you don't need to worry about that. If you are using the results from a form, the server will see the selected color as if the user typed in the hex code manually.

Browser Support

I wanted this to work in all major browsers (even IE). I initially used the canvas tag with an image fallback, but found that CSS gradients were more simple, required less code, and were easier to scale. Browser support depends upon CSS gradient support.

  • IE 6+: Using proprietary filters
  • Chrome 4+
  • Firefox 3.6+
  • Safari 4+
  • Opera 11.1+
  • iOS - tested on an iPhone 3G. If anyone wants to let me know if their device (iPhone or other mobile) works or doesn't, please visit the project page and let me know

Basic Usage

<input type='text' value='red' id='picker' name='picker' />

<script>
$("#picker").spectrum();
</script>

Usage modes

Flat versus Collapsed

  • Flat (see the picker to the left): This means that it will always show up at full size, and be positioned as an inline-block element.

Show Input

  • true (default)
  • false The input that allows free form typing of color from the user can be hidden if desired. If you set opts.showInput to false, it will not show up.

Options (haven't quite finished documenting this part yet)

  • move:event 

    move: function(color) {				
	color.toHexCss(); // #ff0000
}
  • hide:event 

    hide: function(color) {				
	color.toHexCss(); // #ff0000
}
  • show:event 

    show: function(color) {				
	color.toHexCss(); // #ff0000
}
  • beforeShow:event

    Can prevent the colorpicker from showing up if you return false in the beforeShow event.

    beforeShow: function(color) {
	return false;
}

Skinning

Since it is all built with HTML/CSS, you can skin it easily. There are two parts to the spectrum.css file, the core rules (at the top of the file), and the themable rules (at the bottom). Feel free to tweak these rules to make it look how you want.
